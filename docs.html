Docs & replication

Replicate it in one command.

ICS Sentinel runs on the Python standard library alone. No installs, no API key, no hardware. Clone it, run the demo, and watch an AI SOC analyst work a simulated water-treatment plant under attack. Five minutes, start to finish.

$ python -m ics_sentinel.demo

01 · Quickstart

Five steps. Two are optional.

Click copy on any command. The demo never breaks without an API key, it falls back to deterministic [MOCK] triage.

1

Check your Python

You need Python 3.11+. That's the only requirement. The demo has 0 dependencies.

python3 --version
2

Clone the repo

Everything lives in github.com/jamilf/OT-AI.

git clone https://github.com/jamilf/OT-AI.git && cd OT-AI
3

Run the demo

Simulates the plant, injects 5 attacks, detects, maps to ATT&CK, triages, and prints a ranked incident report, in [MOCK] mode if no API key is set.

python -m ics_sentinel.demo
4

Turn on Claude optional

With an Anthropic API key the triage switches to [AI]: live analysis by Claude. rich adds pretty terminal output.

pip install anthropic rich
export ANTHROPIC_API_KEY=sk-ant-…
✓

Verify with the test suite 75 tests

Every number on this site (387 frames, 10 alerts, 0 false positives) is asserted by pytest.

python -m pytest
