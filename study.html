I built myself a study system to prepare properly.

The NSW network assessments test numeracy, reasoning and trade knowledge under time pressure. Rather than cram, I built a spaced-repetition tool to learn the knowledge side and keep it. Every fact in the deck is pinned to a source I can point to, so it never drills me on something wrong. This first deck is electrical knowledge, and it doubles as interview prep.

02 · How it works

Built to be honest with myself.

Spaced repetition (SM-2) method

Each card is scheduled with SM-2, the algorithm behind Anki. Grade a card well and the gap before you see it again grows; miss it and it comes back soon. Reviewing right before you'd forget is what makes knowledge stick, instead of cramming and losing it.

Facts pinned to a source no guessing

Every card carries a source. The deck only contains facts already written up and checked on this site or standard electrical fundamentals. A study tool that quietly teaches you the wrong thing before an exam is worse than no tool, so I built it not to.

What this deck doesn't do yet honest

Spaced repetition is strong for knowledge: formulas, definitions, the isolation sequence. Raw aptitude speed (numeracy and pattern-spotting under the clock) improves mostly through timed volume. A generated, timed aptitude deck is the next thing I want to add; this build is the knowledge half done well first.
